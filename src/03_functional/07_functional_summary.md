# Functional Programming in Python - Summary

Key functional programming concepts:

- **Lambda functions**: anonymous, inline functions
- **Immutability**: favor tuples and strings; avoid side-effects
- **Recursion**: functions calling themselves for iterative logic
- **Higher-order functions**: map, filter, reduce; functions returning functions
- **Pattern matching**: Python 3.10+ match/case for structural patterns

Python supports a functional style alongside OOP and procedural programming.
